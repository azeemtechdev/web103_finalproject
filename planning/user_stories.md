# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

- User: Can create, edit, delete posts and interact with other users through comments and upvotes.

## Draft User Stories

1. As a guest, I want to view all posts so that I can explore content without signing up.

2. As a guest, I want to view a single post so that I can read full details and discussions.

3. As a user, I want to create a post so that I can share ideas or ask questions.

4. As a user, I want to edit my post so that I can update or correct information.

5. As a user, I want to delete my post so that I can remove content I no longer want.

6. As a user, I want to comment on a post so that I can interact with other users.

7. As a user, I want to view comments on a post so that I can follow discussions.

8. As a user, I want to upvote posts so that I can support helpful or interesting content.

9. As a user, I want to filter posts by tags so that I can find content relevant to my interests.

10. As a user, I want to sort posts by newest or most upvoted so that I can quickly find trending or recent posts.

11. As a new user, I want to see default posts when I first use the app so that the platform does not feel empty.

12. As a user, I want to receive error messages when an action fails so that I understand what went wrong.
